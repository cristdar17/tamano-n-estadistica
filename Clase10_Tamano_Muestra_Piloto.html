Motivación

¿Por qué necesitamos calcular n?

El tamaño de muestra no se elige al azar. Un n bien calculado garantiza resultados confiables sin desperdiciar recursos.

🔴

Muestra muy pequeña

Resultados imprecisos, intervalos de confianza amplios, conclusiones poco confiables.

🟢

Muestra adecuada

Precisión óptima, recursos bien utilizados, intervalos de confianza estrechos.

🟡

Muestra muy grande

Mayor costo, más tiempo, desperdicio de recursos sin ganancia significativa de precisión.

💡 Idea clave: Calcular n es encontrar el punto de equilibrio entre precisión estadística y eficiencia de recursos.
Concepto Fundamental

La Muestra Piloto 🧪

Es una muestra pequeña preliminar que se toma de la población antes del estudio principal.

Su propósito: estimar parámetros desconocidos (como σ o p) que necesitamos para calcular el tamaño de muestra definitivo.

⚠️ Problema circular: Para calcular n necesitamos σ, pero para conocer σ necesitamos datos. ¡La muestra piloto rompe este ciclo!
🎯 Definir el objetivo del estudio
↓
🧪 Tomar muestra piloto (n₀ = 20-40)
↓
📊 Calcular s (o p̂) de la piloto
↓
🧮 Usar s (o p̂) para calcular n final
↓
✅ Tomar la muestra definitiva de tamaño n
Fórmula 1

Tamaño de muestra para estimar la Media (μ)

Población infinita o muy grande
n = ( Zα/2 · s / E )²
n Tamaño de muestra requerido
Zα/2 Valor crítico de la distribución normal
s Desviación estándar (estimada de la muestra piloto)
E Error máximo aceptable (margen de error)
Nivel de confianza α α/2 Zα/2
90%0.100.051.645
95%0.050.0251.960
99%0.010.0052.576
Fórmula 2

Tamaño de muestra para estimar una Proporción (p)

Población infinita o muy grande
n = Z²α/2 · p̂ · (1 − p̂) / E²
p̂ Proporción estimada (de la muestra piloto)
1 − p̂ Complemento de la proporción (q̂)
E Error máximo aceptable
Zα/2 Valor crítico (según confianza)
🔑 Caso especial: Si no se tiene muestra piloto ni estimación previa, se usa p̂ = 0.5 (máxima variabilidad → mayor n → escenario más conservador).
Fórmula 3

Corrección por Población Finita

Cuando la población N es conocida y el tamaño de muestra representa más del 5% de la población, se aplica un ajuste.

Fórmula corregida
najustado = n / (1 + (n − 1) / N)

¿Cuándo usar corrección?

Cuando n/N > 0.05 (la muestra es más del 5% de la población).

Empleados de una empresa Estudiantes de una carrera Pacientes de una clínica

¿Cuándo NO usar corrección?

Cuando N es muy grande o desconocido (la población es "infinita" para fines prácticos).

Habitantes de un país Consumidores de un producto masivo
📌 La corrección siempre reduce el tamaño de muestra necesario. Esto tiene sentido: si ya estás muestreando un porcentaje grande de la población, necesitas menos observaciones para obtener la misma precisión.
Ejemplo 1

Estimar la media — Peso de productos 🏭

📋 Una fábrica de cereales quiere estimar el peso promedio de sus cajas de 500g. Se tomó una muestra piloto de 25 cajas y se obtuvo una desviación estándar de s = 12g. ¿Cuántas cajas debe muestrear si quiere un error máximo de 3g con 95% de confianza?
1

Identificar los datos

De la muestra piloto y los requisitos del estudio:

s = 12g  |  E = 3g  |  Confianza = 95% → Zα/2 = 1.960
2

Seleccionar la fórmula

Queremos estimar una media, población grande (infinita para fines prácticos):

n = (Zα/2 · s / E)²
3

Sustituir valores

n = (1.960 × 12 / 3)²
4

Resolver paso a paso

n = (1.960 × 4)²
n = (7.84)²
n = 61.47
5

Redondear hacia arriba (siempre)

El tamaño de muestra siempre se redondea hacia arriba al entero siguiente.

✅ n = 62 cajas
Ejemplo 2

Estimar una proporción — Satisfacción 🏥

📋 Un hospital quiere estimar la proporción de pacientes satisfechos con el servicio. En una muestra piloto de 40 pacientes, 30 se mostraron satisfechos (p̂ = 30/40 = 0.75). ¿Cuántos pacientes debe encuestar con un error máximo de 4% y 95% de confianza?
1

Identificar los datos

p̂ = 0.75  |  q̂ = 1 − 0.75 = 0.25  |  E = 0.04  |  Zα/2 = 1.960
2

Seleccionar la fórmula

n = Z²α/2 · p̂ · q̂ / E²
3

Sustituir valores

n = (1.960)² × 0.75 × 0.25 / (0.04)²
4

Resolver

n = 3.8416 × 0.1875 / 0.0016
n = 0.72030 / 0.0016
n = 450.19
5

Resultado final

✅ n = 451 pacientes
Ejemplo 3

Media con población finita — Horas de estudio 🎓

📋 Una universidad quiere estimar las horas semanales de estudio de sus N = 800 estudiantes de Ingeniería. Una muestra piloto de 30 estudiantes dio s = 5.2 horas. ¿Cuántos estudiantes encuestar con E = 1 hora y 95% de confianza?
1

Datos del problema

N = 800  |  s = 5.2  |  E = 1  |  Zα/2 = 1.960
2

Calcular n sin corrección (población infinita)

n = (Zα/2 · s / E)²
n = (1.960 × 5.2 / 1)²
n = (10.192)²
n = 103.88 → 104
3

Verificar si n/N > 0.05

104 / 800 = 0.13 → 13% > 5% → ¡Sí necesita corrección!
4

Aplicar corrección por población finita

naj = n / (1 + (n − 1) / N)
naj = 104 / (1 + 103/800)
naj = 104 / (1 + 0.12875)
naj = 104 / 1.12875
naj = 92.14
5

Resultado final

La corrección redujo n de 104 a 93. Se ahorran 11 encuestas.

✅ n = 93 estudiantes
Ejemplo 4

Proporción con población finita — Defectos 🔧

📋 Una empresa produjo un lote de N = 2,000 piezas. En una muestra piloto de 50 piezas, se encontraron 8 defectuosas (p̂ = 0.16). ¿Cuántas piezas inspeccionar para estimar la proporción de defectos con E = 3% y 99% de confianza?
1

Datos del problema

N = 2000  |  p̂ = 0.16  |  q̂ = 0.84  |  E = 0.03  |  Zα/2 = 2.576
2

Calcular n sin corrección

n = Z²α/2 · p̂ · q̂ / E²
n = (2.576)² × 0.16 × 0.84 / (0.03)²
n = 6.6358 × 0.1344 / 0.0009
n = 0.8918 / 0.0009
n = 990.93 → 991
3

Verificar corrección

991 / 2000 = 0.4955 → 49.6% > 5% → ¡Necesita corrección!
4

Aplicar corrección

naj = 991 / (1 + (991 − 1) / 2000)
naj = 991 / (1 + 0.495)
naj = 991 / 1.495
naj = 662.88
5

Resultado final

La corrección redujo n de 991 a 663. ¡Se ahorran 328 inspecciones!

✅ n = 663 piezas
Ejemplo 5

Sin muestra piloto — Caso conservador 🛡️

📋 Una empresa de investigación de mercados necesita estimar la proporción de personas que prefieren una marca. No tienen datos previos ni muestra piloto. Quieren un error máximo de 5% con 95% de confianza. ¿Cuántas personas encuestar?
1

Sin piloto → usar p̂ = 0.5 (máxima variabilidad)

Cuando no hay información previa, se asume el peor caso: p̂ = 0.5, que maximiza el producto p̂·q̂ = 0.25.

p̂ = 0.5  |  q̂ = 0.5  |  E = 0.05  |  Zα/2 = 1.960
2

Sustituir en la fórmula

n = (1.960)² × 0.5 × 0.5 / (0.05)²
n = 3.8416 × 0.25 / 0.0025
n = 0.9604 / 0.0025
n = 384.16
3

Resultado

Este es el famoso "n = 385" que aparece en muchas encuestas y estudios de mercado.

✅ n = 385 personas
⚡ ¡Ojo! Con p̂ = 0.5 siempre obtienes el n más grande posible. Si luego descubres que p está lejos de 0.5 (ej. p = 0.1), habrás muestreado de más. La muestra piloto ahorra dinero al dar una estimación más precisa de p.
Ejemplo 6

Comparación de confianza vs tamaño 📈

📋 Usando los datos del Ejemplo 1 (s = 12g, E = 3g), ¿cómo cambia n al variar el nivel de confianza? Comparemos 90%, 95% y 99%. También, ¿qué pasa si cambiamos el error a E = 2g con 95% de confianza?
1

Con E = 3g, variando confianza

90%: n = (1.645 × 12/3)² = (6.58)² = 43.30 → n = 44
95%: n = (1.960 × 12/3)² = (7.84)² = 61.47 → n = 62
99%: n = (2.576 × 12/3)² = (10.30)² = 106.17 → n = 107
2

Con 95% confianza, variando error

E = 5g: n = (1.96 × 12/5)² = (4.704)² = 22.13 → n = 23
E = 3g: n = (1.96 × 12/3)² = (7.84)² = 61.47 → n = 62
E = 2g: n = (1.96 × 12/2)² = (11.76)² = 138.30 → n = 139
E = 1g: n = (1.96 × 12/1)² = (23.52)² = 553.19 → n = 554
3

Conclusiones

📌 Mayor confianza → mayor n (se necesitan más datos para estar más seguro).

📌 Menor error → mayor n (más datos para mayor precisión).

📌 Reducir E a la mitad → cuadruplica n (relación cuadrática inversa).

Resumen

Cheat Sheet — Todo en un vistazo 📋

🎯 Media (pob. infinita)

n = (Zα/2 · s / E)²

s viene de la muestra piloto

📊 Proporción (pob. infinita)

n = Z²α/2 · p̂ · q̂ / E²

p̂ de la piloto, o 0.5 sin datos

🔄 Corrección finita

naj = n / (1 + (n−1)/N)

Aplicar cuando n/N > 5%

💡 Reglas de oro

• Siempre redondear hacia arriba
• Piloto típica: 20–40 observaciones
• Sin datos previos → p̂ = 0.5
• Menor E → n crece cuadráticamente
