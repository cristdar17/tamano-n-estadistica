cap 01 // el problema

Cuando dudamos…

🪙 ¿Esa moneda con la que tu amigo siempre gana está cargada?
💊 ¿Ese medicamento nuevo de verdad funciona… o es lo mismo de siempre?
📸 ¿La gente de verdad distingue una foto con filtro de una sin filtro?

Toda la estadística inferencial responde estas preguntas con el mismo método.

El método: H₀ vs H₁

► HOY LO APRENDES ◄

cap 02 // las dos rivales

Toda investigación enfrenta 2 hipótesis rivales

// hipótesis nula

H₀

"Aquí no pasa nada."

Es lo aburrido, lo de siempre. La afirmación de que nada cambió, que la diferencia que vemos es por puro azar.

// hipótesis alternativa

H₁

"Sí pasa algo."

Es la sospecha que queremos demostrar. La afirmación de que sí hay diferencia, que esto no es solo azar.

🕵️ Como un detective: H₀ es "no pasó nada raro", H₁ es "sí hubo crimen".
La pregunta NO es "¿quién tiene la razón?"
La pregunta es: ¿hay suficiente evidencia para abandonar H₀?

// REGLAS PARA ESCRIBIRLAS

REGLA #1

H₀ siempre lleva el signo = (igualdad)

REGLA #2

H₁ lleva >, < o ≠ (desigualdad)

// EJEMPLOS RÁPIDOS

PREGUNTA H₀ H₁
¿La moneda es justa?p = 0.5p ≠ 0.5
¿El gym hace bajar de peso?μ = 0μ > 0
¿El medicamento A cura mejor?μ_A = μ_Bμ_A > μ_B
¿Llueve más los lunes?μ_lun = μ_restoμ_lun ≠ μ_resto

Esa "rareza" tiene un nombre: p-value

cap 04 // el p-value

P-value: "qué tan raro sería esto si H₀ fuera cierta"

Imagina que H₀ es VERDAD (la moneda sí es justa). Vamos a simular 10.000 experimentos de "tirar la moneda 100 veces" y ver cuántas caras salen en cada uno.
► cada partícula = 1 experimento completo de 100 tiros
simulaciones: 0 media: — tu observación: — p-value: —

↳ Cómo leerlo

p < 0.05

"Lo que vimos sería rarísimo si H₀ fuera cierta."
► RECHAZAMOS H₀

p ≥ 0.05

"Lo que vimos podría perfectamente ocurrir bajo H₀."
► NO RECHAZAMOS H₀

⚠️ Cuidado: "no rechazar H₀" NO es lo mismo que "aceptar H₀".
simplemente significa: la evidencia no alcanzó.
cap 05 // errores tipo I y II

Podemos equivocarnos de 2 formas

Incluso haciendo todo bien, nuestra decisión puede salir mal por puro azar. Hay dos errores posibles, y los llamamos Tipo I y Tipo II. Pasa el cursor por cada cuadrante.

// ERROR TIPO I = α

Rechazar H₀ cuando es verdadera.

Es el riesgo que nosotros fijamos. Convención: α = 0.05 (5%).

// ERROR TIPO II = β

No rechazar H₀ cuando es falsa.

Depende del tamaño de muestra y del efecto real. Más datos → menos β.

🎯 Idea clave: bajar α (ser más estricto) aumenta β. Y viceversa.
no se pueden minimizar los dos a la vez con datos fijos
cap 06 // veredicto

El detective junta evidencia → llega el veredicto

Cada nueva observación suma a la barra de evidencia. Cuando esa evidencia cruza el umbral del 95% (o sea, p < 0.05), el detective tiene caso: rechaza H₀.
datos recolectados: 0 fuerza de evidencia: 0% p-value estimado: —

VEREDICTO: RECHAZAR H₀

La evidencia es suficientemente rara bajo H₀. Aceptamos la sospecha (H₁).

// LA RECETA COMPLETA

  1. 1. Plantea H₀ y H₁
  2. 2. Recolecta datos
  3. 3. Calcula el p-value
  4. 4. Compara con α (típico 0.05)
  5. 5. Decide: rechazar o no rechazar H₀
FIN PARTE 1

► AHORA: 5 EJEMPLOS DEL MUNDO REAL ◄

ejemplo 01 / 05
🪙

¿La moneda de tu amigo está cargada?

Tu amigo tira la moneda 10 veces y le sale 7 caras (gana 7 apuestas seguidas). ¿Es trampa o azar normal?
// hipótesis nula
H₀ : p = 0.5
La moneda es justa, lo que pasó fue suerte.
// hipótesis alternativa
H₁ : p ≠ 0.5
La moneda está cargada hacia caras (o sellos).

// CÁLCULO

Si la moneda fuera justa, ¿qué tan raro es ver 7 o más caras en 10 tiros?
Probabilidad (binomial): 0.172

// VEREDICTO
NO RECHAZAMOS H₀
p = 0.172 > 0.05 → 7 caras en 10 tiros pasaría el 17% de las veces solo por azar. No hay evidencia para acusarlo.
🤔 Pregunta extra: ¿y si fueran 70 caras en 100 tiros? Misma proporción, pero ahora p = 0.0001 → RECHAZAMOS H₀.
↳ moraleja: el tamaño de la muestra cambia todo.
ejemplo 02 / 05
☕

¿El café de la esquina tiene más cafeína?

Juan Valdez tradicional tiene 95 mg de cafeína por taza (μ₀). El café de la esquina afirma tener más. Mides 30 tazas y promedias 110 mg con desv. estándar 20 mg.
// hipótesis nula
H₀ : μ = 95 mg
El café de la esquina tiene la misma cafeína que Juan Valdez.
// hipótesis alternativa
H₁ : μ > 95 mg
El café de la esquina tiene más cafeína (de un solo lado).
JUAN VALDEZ (μ₀)
95 mg
EL DE LA ESQUINA
110 mg ± 20

// CÁLCULO (test t)

t = (110 − 95) / (20 / √30) = 4.11

Para 29 grados de libertad y test de una cola: p ≈ 0.00014

// VEREDICTO
RECHAZAMOS H₀
p ≈ 0.00014 << 0.05 → es prácticamente imposible obtener este resultado si fueran iguales. Sí: el café de la esquina tiene significativamente más cafeína.
ejemplo 03 / 05
☔

¿De verdad llueve más los lunes en Pereira?

Tu mamá jura que los lunes llueve más en Pereira. Recoges datos de 52 lunes y 52 días promedio del resto de la semana durante un año.
// hipótesis nula
H₀ : μ_lun = μ_resto
Llueve igual los lunes que cualquier otro día.
// hipótesis alternativa
H₁ : μ_lun ≠ μ_resto
Llueve diferente los lunes (más o menos).

// MM DE LLUVIA PROMEDIO POR DÍA (año completo)

// CÁLCULO (test t para 2 muestras)

Lunes: μ̄ = 12.4 mm | Resto: μ̄ = 10.1 mm | Diferencia = 2.3 mm

t = 1.34 → p ≈ 0.18

// VEREDICTO
NO RECHAZAMOS H₀
p ≈ 0.18 > 0.05 → la diferencia de 2.3 mm es perfectamente compatible con azar. Lo siento mamá: con estos datos, los lunes no llueve significativamente más.
💡 Sesgo de confirmación: recordamos los lunes lluviosos (porque odiamos los lunes) y olvidamos los soleados. Por eso necesitamos datos, no intuición.
ejemplo 04 / 05
💪

¿El nuevo gym funciona o es placebo?

Un gym promete bajar peso. 40 personas entrenan 3 meses. Promedio: bajaron 2.1 kg con desv. estándar 3.2 kg. ¿Esto es por el gym… o pasaría igual sin entrenar?
// hipótesis nula
H₀ : μ = 0 kg
El gym no hace nada, el cambio promedio es 0.
// hipótesis alternativa
H₁ : μ > 0 kg
El gym sí hace bajar peso (en promedio).

// CADA PUNTO = 1 PERSONA · KG PERDIDOS

KG ↑ PARTICIPANTES (1—40)

// CÁLCULO

t = 2.1 / (3.2 / √40) = 4.15

Para 39 grados de libertad y test de una cola: p ≈ 0.00009

// VEREDICTO
RECHAZAMOS H₀
p ≈ 0.00009 << 0.05 → bajar 2.1 kg en promedio con estos datos sería casi imposible si el gym no hiciera nada. El gym sí funciona (al menos para esta muestra).
⚠️ OJO: "estadísticamente significativo" ≠ "importante en la vida real". 2.1 kg en 3 meses puede ser un cambio modesto. Significancia estadística es solo "no es azar".
ejemplo 05 / 05
📸

¿La gente distingue filtro de Instagram?

Test ciego: a 200 personas les muestras 2 fotos del mismo paisaje (una real, una con filtro). Si fueran indistinguibles (H₀), esperaríamos que el 50% acertara cuál es real. En realidad, 128 acertaron (64%).
// hipótesis nula
H₀ : p = 0.5
No se distingue. Aciertan al azar (puro 50/50).
// hipótesis alternativa
H₁ : p > 0.5
Sí se nota el filtro: aciertan más del 50%.

// LAS DOS FOTOS

REAL🏞️
FILTRO✨

resultado: 128 / 200 = 64% acertaron cuál era real

// CÁLCULO (test z para proporción)

z = (0.64 − 0.50) / √(0.5·0.5/200) = 3.96

Test una cola: p ≈ 0.00004

// VEREDICTO
RECHAZAMOS H₀
p ≈ 0.00004 << 0.05 → es prácticamente imposible que 128 de 200 acertaran por azar. Sí, la gente nota el filtro (más de lo que pensábamos).
FIN DE LA CLASE

H₀ vs H₁ es el método.
Lo demás (test t, test z, ANOVA, chi-cuadrado…) es solo cómo calcular el p-value en cada caso.
