Instrucciones

Ahora es en serio: los datos vienen en bruto 💪

📊

1. Lee los datos crudos

Cada ejercicio te da los datos reales de la muestra piloto. Números o respuestas Sí/No tal como salieron.

🧮

2. Calcula la variabilidad

Para medias: calcula x̄ y s (desviación estándar). Para proporciones: cuenta éxitos y calcula p̂.

📐

3. Aplica la fórmula de n

Con tu s o p̂ calculado, aplica la fórmula correcta. Verifica con el botón de respuesta.

Recordatorio de fórmulas
Media: n = (Z · s / E)²    |    Proporción: n = Z² · p̂ · q̂ / E²
⚠️ Recuerda: s = √[Σ(xi − x̄)² / (n−1)] — Y si n/N > 5%, aplica corrección finita: naj = n / (1 + (n−1)/N)

Fórmula: n = (Z · s / E)²

Cierre

Lo que practicaste hoy 🎯

🧮 Calcular s desde datos crudos

Ya no dependes de que alguien te diga s = tal. Sabes tomar datos, calcular x̄, obtener las desviaciones, elevar al cuadrado, sumar, dividir por n−1 y sacar raíz.

📊 Calcular p̂ contando

Para proporciones, lo único que necesitas es contar éxitos y dividir entre el total. Sin eso, no puedes calcular n.

🔍 Detectar cuándo corregir

Calculaste n, verificaste n/N, y solo corregiste cuando superó el 5%. Esa verificación ya es automática en tu proceso.

🏢 Flujo completo de la vida real

Datos crudos → calcular variabilidad → elegir fórmula → sustituir → verificar finita → redondear. Así se hace en una empresa.

🔥 La diferencia entre un estudiante que solo aplica fórmulas y uno que sabe obtener los datos para la fórmula es la diferencia entre alguien que sigue instrucciones y alguien que resuelve problemas. Hoy diste ese salto.
