Aprendizajes Clave

  • Para variables cuantitativas, se usa la desviacion estandar (σ) en la formula del tamano de muestra.
  • Para variables cualitativas, se usan las proporciones p y q = 1-p.
  • La correccion por poblacion finita (n = n₀ / (1 + n₀/N)) siempre reduce el tamano de muestra requerido.
  • A mayor nivel de confianza (Z mayor), se necesita una muestra mas grande.
  • A menor margen de error (e), se necesita una muestra mas grande.
  • Cuando N es muy grande, la correccion finita tiene poco efecto y n se acerca a n₀.
